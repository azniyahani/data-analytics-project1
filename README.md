# Data Analytics Project 1 - Data Cleaning Using Python

## Project Overview

This project was completed as part of a Data Analytics Internship task.

The objective of this project was to clean and analyze an Excel dataset using Python and pandas.

---

## Tasks Performed

* Loaded Excel dataset using pandas
* Displayed dataset information
* Checked missing values
* Checked duplicate rows
* Checked duplicate Order IDs
* Converted Date column into proper datetime format
* Filled missing CouponCode values
* Removed duplicate rows
* Saved cleaned dataset into a new Excel file
